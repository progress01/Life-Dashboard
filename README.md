# Life-Dashboard
## 簡介
日常生活關鍵指標的儀表板。透過連結的數據來源（熱量追蹤和記帳），幫助維持連續紀錄並達成目標。

## 核心功能與儀表板元件
*   ** 熱量 (Calories)：** 追蹤和管理您的熱量攝取與消耗。
*   ** 記帳 (Accounting)：** 管理財務狀況，並能顯示「**錢包 HP (Wallet HP)**」等指標。
*   ** 獎勵 (Rewards)：** 透過遊戲化的方式提供熱量獎勵（🔥 熱量獎勵）和消費獎勵（💰 消費獎勵）。

## 數據追蹤與遊戲化指標

儀表板提供多項即時指標和追蹤功能，鼓勵使用者維持習慣：

1.  **即時狀態與預算：** 顯示「錢包 HP」（Wallet HP）及「今日理論剩餘」的數值。
2.  **連續性追蹤：** 記錄「連續紀錄 (天)」和「總紀錄天數」。
3.  **熱力圖 (Heatmap)：** 使用者可以透過「👆 滑動熱力圖」來查看詳細的紀錄與數據詳情。
4.  **獎勵系統：** 儀表板顯示「該週累積成果」，並提供「🎰 抽取獎勵」的功能。使用者甚至可以自行「👇 編輯此模式的獎品清單」（需一行一個輸入獎品）。

## ⚙️ 設定與資料來源

本儀表板設計為高度客製化，主要的數據來源依賴外部試算表（例如 Google Sheets 或其他連結），使用者需要在設定中提供特定資訊：

| 設定項目 | 描述 | 來源格式要求 |
| :--- | :--- | :--- |
| **熱量資料** | 設定熱量追蹤數據的連結。 | 需提供熱量表連結（或 ID）及熱量表分頁名。 |
| **記帳資料** | 設定財務記帳數據的連結。 | 需提供記帳表連結（或 ID）及記帳表分頁名。 |
| **財務預算** | 設定每月的記帳追蹤上限。 | 需設定「記帳月預算」。 |
| **金句設定** | 設定用於顯示每日激勵話語或金句的分頁。 | 需提供「金句分頁名」。 |

## 如何開始
1.  進入 ⚙️ 資料來源設定頁面。
2.  填寫您熱量表、記帳表及金句的連結/ID 與分頁名。
3.  設定您的「記帳月預算」。
4.  點擊「💾 儲存設定」完成配置。

## Google Sheet 表單的設定方式與程式對接

高度依賴外部 Google Sheet 資料源運作的系統。為了使程式能夠讀取並呈現您的數據，在儀表板的「資料來源設定」頁面中提供三組主要的資料連結與名稱。

### 1. 資料來源設定的步驟

您需要先建立或準備好2個獨立的 Google Sheet 分頁（Tabs），然後在儀表板的設定介面中輸入相關資訊：

| 設定項目 | 描述 | 來源格式要求 |
| :--- | :--- | :--- |
| **熱量表連結 (或 ID)** | 這是您追蹤飲食和熱量數據的試算表連結或 ID。 |
| **熱量表分頁名** | 該試算表中，專門儲存熱量數據的分頁名稱。 |
| **記帳表連結 (或 ID)** | 這是您追蹤財務記帳數據的試算表連結或 ID。 |
| **記帳表分頁名** | 該試算表中，專門儲存記帳數據的分頁名稱。 |

在輸入完這些資訊後，您需要點擊「儲存設定」來完成配置。

### 2. 欄位對應與程式對接的說明
